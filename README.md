# FocusFlow

Десктопное приложение для управления продуктивностью. Построено на C# / Avalonia UI / .NET 8, паттерн MVVM (CommunityToolkit.Mvvm), локальное хранилище LiteDB, облачная синхронизация через Supabase.

## Реализованные функции

### Задачи
- Создание, редактирование, удаление задач
- Приоритет, дата выполнения, время начала/окончания, плановая длительность
- Подзадачи с отслеживанием прогресса
- Группировка по проектам (до 3 проектов в бесплатном плане)
- Лимиты для гостевого режима: 50 задач / 3 проекта / 50 событий

### Календарь
- Представления: День, Неделя, Месяц, Год
- Подсветка сегодняшнего дня во всех видах
- Виртуальные события из задач с датами
- Значок заметок на днях с записями

### События
- Создание событий с уведомлениями (за 10 мин / 30 мин / 1 час / 1 день)
- Повторяющиеся события (ежедневно / еженедельно / ежемесячно / ежегодно)
- Копирование события на несколько дат (мини-календарь с мультиселектом)

### Pomodoro-таймер
- Таймер с кастомными шаблонами сессий
- Диалог завершения с итогами
- Автоматическое завершение связанных привычек по окончании таймера

### Трекер привычек
- Список привычек с ежедневной отметкой (выполнено / частично)
- GitHub-style тепловая карта (12 недель)
- Месячная сетка с 3-состоянием (нажатие: нет → выполнено → частично)
- Стрики: текущий и лучший; ачивменты (7/30/90 дней, 100 выполнений)
- Категории и иконки
- Шаблоны привычек (системные + пользовательские)
- Интеграция с задачами: автоотметка привычки при завершении связанной задачи
- Экспорт истории в CSV и Excel

### Заметки / Дневник
- Markdown-редактор с панелью форматирования (жирный, курсив, код, заголовки, цитата, список)
- Автосохранение с задержкой 4 секунды
- Теги, поиск по тексту и тегам
- Значки заметок на днях в календаре (Месяц-вид)
- Экспорт в Markdown / HTML / Plain Text

### Тренировки
- Программы тренировок и журнал сессий
- Упражнения с метриками: вес+повторения / только повторения / только время / время+повторения
- Таймер отдыха между подходами
- Статистика по году (аналитика, защита от краша на пустом году)

### Финансы
- Доходы, расходы, подписки, кредиты/займы
- Счета и копилки с целевой суммой и прогресс-баром
- Автоматическая проверка достижения цели копилки
- Сводная вкладка с балансом и статистикой
- Удаление пользовательских категорий с nullification затронутых записей
- Экспорт всех данных в Excel (.xlsx, ClosedXML)

### Аналитика
- Недельная навигация (стрелки вперёд/назад)
- Сводка по году тренировок

### Фоновые звуки
- Пресеты природных звуков (LibVLCSharp)
- Воспроизведение в фоне во время работы

### Уведомления
- Системные уведомления через Avalonia WindowNotificationManager
- Поллинг каждую минуту; срабатывают за заданное время до события

### Темы
- Фиксированная система тем: Светлая / Тёмная / Розовая / Красная / Жёлтая / Зелёная / Синяя
- Тема применяется только по кнопке «Сохранить»
- Полная поддержка тёмного режима (иконки, тайлы, кнопки)

### Аккаунт и синхронизация
- Регистрация и вход через Supabase Auth
- Облачная синхронизация данных (SupabaseSyncService)
- Модель владельца данных: `"local"` (гость) или UserId (аккаунт)
- При выходе из аккаунта: синхронизация → очистка сессии → переключение на локальный режим
- При превышении лимитов (гостевой режим): диалог выбора записей для удаления
- Управление подпиской (300 ₽/мес, 2 000 ₽/год)

### Локализация
- Русский и английский интерфейс
- Переключение без перезапуска приложения

## Настройка

Скопируй `appsettings.example.json` → `appsettings.json` и вставь свои ключи:

```json
{
  "Supabase": {
    "Url": "YOUR_SUPABASE_URL",
    "AnonKey": "YOUR_SUPABASE_ANON_KEY"
  },
  "YandexMaps": {
    "SuggestApiKey": "",
    "GeocoderApiKey": "",
    "StaticApiKey": ""
  }
}
```

`appsettings.json` добавлен в `.gitignore` — реальные ключи не попадут в репозиторий.
